# Config Provide 全局配置

全局配置组件的尺寸、主题、字体大小等

## Props

| 属性名       | 说明           | 类型                         | 默认值   |
| ------------ | -------------- | ---------------------------- | -------- |
| htmlFontSize | 根节点字体大小 | number                       | 16       |
| size         | 组件尺寸       | `small` / `medium` / `large` | `medium` |
| theme        | 样式主题       | `browser` / `ustyle`         | `ustyle` |
