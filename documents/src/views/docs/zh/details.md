# Details 详细信息展现元素

仅在被切换成展开状态时，它才会显示内含的信息

## 基础用法

::: demo
<uDetails>
<template #summary>详情</template>
正文内容。
</uDetails>
:::

## Props

| 属性名 | 说明         | 类型    | 默认值 |
| ------ | ------------ | ------- | ------ |
| open   | 内容是否可见 | boolean | false  |

## 插槽

| 插槽名  | 说明 | Solot Props |
| ------- | ---- | ----------- |
| default | 内容 | -           |
| summary | 概要 | -           |

## 事件

| 事件名 | 说明                       | 回调参数           |
| ------ | -------------------------- | ------------------ |
| change | 状态在打开和关闭之间变化时 | `(value: boolean)` |
