# Config Provide: Global Configuration

Provides global configuration for component size, theme, font size, and more.

## Props

| Prop Name    | Description    | Type                         | Default  |
| ------------ | -------------- | ---------------------------- | -------- |
| htmlFontSize | Root font size | number                       | 16       |
| size         | Component size | `small` / `medium` / `large` | `medium` |
| theme        | Theme style    | `browser` / `ustyle`         | `ustyle` |
